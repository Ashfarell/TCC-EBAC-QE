# 🧪 Projeto de Automação de Testes - TCC QA

Este projeto foi desenvolvido como parte do Trabalho de Conclusão de Curso (TCC) da formação de Quality Assurance Engineer.

O objetivo é validar funcionalidades de uma aplicação web e uma API utilizando testes automatizados.

---

# 📌 Objetivo

Automatizar testes de:

- Interface Web (UI) com Cypress
- API REST com Jest + Supertest

---

# 🧰 Tecnologias utilizadas

- Cypress
- Jest
- Supertest
- Node.js
- JavaScript
